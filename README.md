## Description
This webpage is a mock up of the Andersson-wise website landing page. It uses Mondrian Design pattern make content responsive.
Relative units are used where appropriate.
Page respond to three different screen sizes.
Makes use of media queries, no bootstrap.
Images are fluid.
Demonstrates understanding of responsive design patterns.

## Setup/Installation Requirements

* Set up HTML
* Link css styles page
* Add content
* No extensive setup
